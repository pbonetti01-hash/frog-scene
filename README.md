# 🐸 Frog Pond Scene - Unity 3D

A serene, stylized 3D environment created in Unity featuring cute, low-poly frogs in a vibrant pond setting. This project showcases custom textures, custom shaders, hand-crafted particle effects, and curated asset store integration.

---

## 🛠️ Project Details & Specifications

* **Game Engine:** Unity (3D)
* **Art Style:** Stylized / Low-Poly / Ghibli-inspired
* **Texturing Software:** Krita (All environment and character textures are 100% hand-painted/custom-made)
* **Skybox Asset:** *Fantasy Skybox FREE* (Unity Asset Store)

## ✨ Key Features & Components

### 🎨 1. Custom Textures (Hand-painted in Krita)
* **Frogs:** Soft, organic green gradients giving them a cute, friendly, and stylized appearance.
* **Water Lilies & Reeds:** Unique hand-drawn details that complement the low-poly meshes.
* **Water Surface:** Textured to simulate a vibrant, crystal-clear stylized pond with beautiful reflection/refraction tones.

### 🧪 2. Custom Shaders (Built in Unity)
The project utilizes **two custom shaders** developed from scratch to achieve the unique artistic look:
1.  **Stylized Water Shader:** * Features a custom pattern for the caustics/ripples.
    * Includes soft edge-fading (depth-based foam) where the water meets the reeds and lily pads.
    * Dynamic vertex displacement to create gentle, calming wave movements.
2.  **Stylized Environment/Character Shader:**
    * A custom lighting/toon shader designed to bring out the hand-painted details from Krita without relying on realistic PBR lighting.
    * Maintains vibrant, saturated colors matching the overall dream-like aesthetic.

### 🌧️ 3. Particle Systems
* **Custom Rain System:** A lightweight, optimized Unity Particle System designed to create soft, atmospheric rain droplets that fall over the pond, enhancing the cozy mood of the environment.

### 🌌 4. Skybox
* **Asset Used:** *Fantasy Skybox FREE* from the Unity Asset Store.
* Features fluffy, anime-styled cumulus clouds and a brilliant blue gradient sky that blends seamlessly with the pond's color palette.
